# Scripts Directory

Эта папка содержит различные вспомогательные скрипты для управления проектом.

## Мониторинг отключений электроэнергии

- **setup-telegram.sh** - Интерактивная настройка Telegram бота для уведомлений
- **test-power-outage.sh** - Проверка системы мониторинга отключений

## База данных

- **db-manager.sh** - Управление базой данных
- **import-db.sh** - Импорт базы данных
- **import-db-interactive.sh** - Интерактивный импорт БД
- **test-db-tools.sh** - Тестирование инструментов БД

## API и интеграции

- **fetch-all-api-data.sh** - Получение всех данных из API
- **webhook-setup-help.sh** - Помощь по настройке вебхуков

## Docker и сервисы

- **docker-manager.sh** - Управление Docker контейнерами
- **start-ngrok.sh** - Запуск ngrok для локальной разработки

## Хранилище

- **setup-storage.sh** - Настройка хранилища Laravel

## Тестовые скрипты PHP

- **assign-role.php** - Назначение ролей пользователям
- **test-badge.php** - Тестирование бейджей
- **test-csv-encoding.php** - Тестирование кодировки CSV
- **test-import-duplicates.php** - Тестирование дубликатов при импорте
- **test-password-reset.php** - Тестирование сброса пароля

## Использование

Все скрипты должны быть исполняемыми. Если нет, выполните:

```bash
chmod +x scripts/*.sh
```

Запуск скрипта:

```bash
./scripts/имя-скрипта.sh
```

Или из корня проекта:

```bash
bash scripts/имя-скрипта.sh
```

## Примечание

PHP скрипты запускаются через:

```bash
php scripts/имя-скрипта.php
```
